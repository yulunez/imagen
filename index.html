Acentos del español


El acento es la mayor intensidad con que se pronuncia una de las sílabas de una palabra.
La tilde o acento ortográfico es la marca escrita que se coloca sobre una letra para indicar que esa sílaba se pronuncia con mayor intensidad.
Las mayúsculas también se escriben con tilde.

  2. Esdrújula
